## Description
In the final exam we were tasked with creating a accommodation booking site called Holidaze. All the way from planning the project with a kanban and ghantt board for issues, and timing, to desiging the prototype in figma and lastly develop and deploy the page.
The allotted time given was 7 weeks, with a set of user stories that are required to be implemented.

Exam's key points:
- [Prototyping with Figma](https://www.figma.com/design/VL7PdkHBvVBj5BRxwLicqM/Project-Exam-2?node-id=1-2&t=j09TppWPS101Sjy9-0) (My Prototype)
- Coding required user stories:
  - A user may view a list of Venues
  - A user may search for a specific Venue
  - A user may view a specific Venue page by id
  - A user may view a calendar with available dates for a Venue
  - A user with a stud.noroff.no email may register as a customer
  - A registered customer may create a booking at a Venue
  - A registered customer may view their upcoming bookings
  - A user with a stud.noroff.no email may register as a Venue manager
  - A registered Venue manager may create a Venue
  - A registered Venue manager may update a Venue they manage
  - A registered Venue manager may delete a Venue they manage
  - A registered Venue manager may view bookings for a Venue they manage
  - A registered user may login
  - A registered user may update their avatar
  - A registered user may logout
- Responsive design that functions on most devices and screens.
- Set up the page for a deployed version on Netlify.

## Getting Started

### Installing

1. Clone the repo:

```bash
https://github.com/ESPR07/project-exam-2.git
```

2. Install all required dependencies:
```bash
npm install or yarn install (depending on which you use)
```

3. then run the project locally with:
```bash
npm run dev or yarn dev (depending on which you use)
```
